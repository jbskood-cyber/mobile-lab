# Siguientes pasos

1. Integrar la base técnica en `main`.
2. Clonar el repositorio en la computadora del usuario.
3. Ejecutar `npm install`.
4. Ejecutar `npm run start:lan`.
5. Abrir el QR con Expo Go en el Samsung.
6. Confirmar la pantalla temporal.
7. Crear una rama `feature/foco-foundation` para comenzar el sistema visual y la navegación de FOCO.
